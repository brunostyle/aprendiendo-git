  - **git log**

  - **git log -[numero]**

  - **git log --author=[autor]**  
    Muestra solo commits hechos por un autor espec칤fico.

    Lista todas las ramas.
  - **git branch -v**  
    Muestra el 칰ltimo commit de cada rama local.
  - **git branch -d / -D [nombre rama]**
